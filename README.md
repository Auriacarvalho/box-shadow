# Box Shadow - CSS

## 📌 Descrição
Este projeto foi desenvolvido para praticar o uso da propriedade **box-shadow** do CSS, criando efeitos visuais modernos em elementos da interface.  
O objetivo é demonstrar como sombras podem ser aplicadas para dar profundidade e estilo a componentes web.

## 🚀 Tecnologias Utilizadas
- **HTML5** → estrutura da página  
- **CSS3** → estilização e aplicação de sombras  

## 📂 Estrutura do Projeto
- `index.html` → página principal  
- `style.css` → estilos aplicados com box-shadow  
- `assets/` → imagens e ícones (se houver)  

## 📊 Funcionalidades
- Demonstração de diferentes tipos de sombras em caixas e botões  
- Efeitos visuais responsivos e modernos  
- Layout simples para prática de CSS
